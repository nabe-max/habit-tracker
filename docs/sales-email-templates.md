> Instagram は更新してるのに HP だけ古い、も **⑨** が刺さりやすい。

**件名（代替）：**
- 【ご提案】【店名】様のWebサイト更新について
- 【店名】様のWebサイトを拝見しました


### 観察の書き換え例（更新停止）

**必ず「褒め1行 → 更新の観察1行」の順で。**

| 見つけた状態 | やんわり書き方 |
|-------------|----------------|
| お知らせが数年前 | 最新のお知らせが【○年】の内容のままで、現在の情報が伝わりにくいかもしれません |
| メニュー・料金が古い | メニューや料金表が、現在の内容と一致していない可能性がありそうです |
| 営業時間・定休日 | 営業時間や定休日の情報が、最新と異なっているかもしれません |
| ブログ・コラム停止 | ブログやコラムの更新が止まっており、サイト全体が止まっている印象を与えているかもしれません |
| Instagram は更新中 | Instagramでは活発に発信されている一方、Webサイトの情報が追いついていないように感じました |
| 季節・キャンペーン情報 | 季節限定メニューやキャンペーン情報が、過去の内容のまま残っているようです |
| スタッフ・代表者情報 | スタッフ紹介や代表挨拶など、古い情報が残っている箇所がありそうです |

**注意：**
- 「放置してます」「更新してないですね」とは書かない
- 必ず **事実ベース**（「2021年の記事が最新」など）にする
- Instagram と比較する場合は、**Instagram を褒める** トーンで
